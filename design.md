This design system theme uses a **light color mode**.

**Colors:**
The primary brand color is `#25aff4`, supported by a secondary `#557b97` and an accent tertiary `#e3940a`. Neutral elements are based on `#73787d`.

**Typography:**
All text elements (headlines, body, and labels) use the `Inter` font.

**Shape:**
UI elements feature a subtle roundedness, indicating a modern but not overly soft aesthetic.

**Spacing:**
The layout maintains a normal level of spacing, providing a balanced and comfortable visual experience.
